**Technology stack**
* *API*: Aiogram

# Installation:
1) Clone this repo: `git clone https://github.com/ornarasus/Woordly.git`
2) Install package: `pip install aiogram`
3) Setup token of telegram bot in `settings.py`
4) Run: `python run.py`
